# SmartFactory — Browser-Lesezeichen (Import)

Eine Datei: **[SmartFactory.html](./SmartFactory.html)** — enthält den Ordner **SmartFactory** mit allen Unterordnern (OSF-RPi, OSF-LH, MES, DSP, fischertechnik, Dev). Keine privaten Favoriten außerhalb.

**Verwendung:** [windows-desktops-teams-obs-setup-checklist.md](../windows-desktops-teams-obs-setup-checklist.md)

Tab-Titel und Reihenfolge wie im Kollegen-Export. **OSF-URLs** auf Locale **`/de`** vereinheitlicht.

## Import (Edge / Chrome)

1. **Einmal** [SmartFactory.html](./SmartFactory.html) importieren (**Favoriten importieren** / **Lesezeichen importieren**) — in **Edge** und **Chrome**.
2. Ergebnis: Favoritenordner **SmartFactory** mit allen Unterordnern.
3. Tab-Gruppen: Unterordner → **Alle Tabs in Tabgruppe öffnen** (Edge/Chrome, siehe Checkliste).

| Browser | Import |
|---------|--------|
| **Edge** | **Favoriten** → **Favoriten verwalten** → **…** → **Favoriten importieren** |
| **Chrome** | **Lesezeichen-Manager** (`Strg+Umschalt+O`) → **⋮** → **Lesezeichen importieren** |
