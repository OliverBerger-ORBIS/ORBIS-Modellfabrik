Data-Basis Simulation

This demo focuses on collecting condition data streams (vibration, flame, gas, temperature/humidity) as predictive-maintenance input. Immediate anomaly escalation is covered in UC-07.

CCU limitation: Demo commands can affect only ENQUEUED orders. Running orders continue until completion.

Demo steps

  1. Activate the toggle above
  2. Trigger order in Process tab
  3. Generate sensor data events (vibration, flame, gas, or temp/hum)
  4. Observe data flow and context correlation in Message Monitor
