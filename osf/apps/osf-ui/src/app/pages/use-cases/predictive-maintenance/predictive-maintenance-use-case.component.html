Danger Simulation

When enabled, a vibration alarm (RED) triggers park + cancel of ENQUEUED orders automatically. Use the Park button in the header for manual park.

Demo steps

  1. Activate the toggle above
  2. Trigger order in Process tab
  3. During order process, generate vibration (impact or tuning fork on sensor)
