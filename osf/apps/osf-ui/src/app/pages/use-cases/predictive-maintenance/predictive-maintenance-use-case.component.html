Danger Simulation

When enabled, any Arduino sensor alarm (vibration, flame, gas, temperature/humidity) triggers park + cancel of ENQUEUED orders automatically. Use the Park button in the header for manual park.

CCU limitation: Only ENQUEUED orders are cancelled. Orders already IN_PROGRESS (e.g. at DRILL or AIQS) continue until completion. Manual factory reset may be required after alarm.

Demo steps

  1. Activate the toggle above
  2. Trigger order in Process tab
  3. During order process, trigger any sensor alarm (vibration, flame, gas, or temp/hum)
  4. Observe order flow in Orders tab
